# Website Project

This repository contains the HTML files for a simple website. The project includes the following pages:

- `index.html`
- `contact.html`
- `portfolio.html`
- `products.html`
- `proposal.html`

## Overview

This is a basic static website built with HTML. You can open any of the above files in a web browser to view the corresponding page.

## Usage

1. Clone or download the repository.
2. Open one of the HTML files in your browser (e.g., double-click `index.html`).
